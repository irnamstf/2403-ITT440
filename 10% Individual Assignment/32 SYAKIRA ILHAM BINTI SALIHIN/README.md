# HOW TO COMPILE A SIMPLE C PROGRAM

The process of translating the C language's source code into machine code is called compilation. Considering C is a mid-level language, in order for a programÂ to operate on our machine, it must be translated into an executable code using a compiler.

We first need a compiler and a code editor to compile and run a C Program. The below example is of MobaXterm Apps with GCC compiler.
